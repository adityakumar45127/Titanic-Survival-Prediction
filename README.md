# Titanic Survival Prediction

## Project Overview

This project is a Machine Learning web application that predicts whether a passenger would survive the Titanic disaster based on passenger information.

The application is built using Python, Scikit-learn, and Streamlit.

---

## Features

- Predict Titanic passenger survival
- Data preprocessing using Scikit-learn Pipeline
- Random Forest Classifier
- Interactive Streamlit Web App
- Clean Machine Learning Pipeline

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Run Locally

Clone the repository

```bash
git clone https://github.com/adityakumar45127/Titanic-Survival-Prediction.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit app

```bash
streamlit run APP.py
```
